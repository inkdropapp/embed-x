# Embed X (Twitter) plugin for Inkdrop

Embed X (formerly Twitter) posts directly in your Inkdrop notes.

## Install

```sh
ipm install embed-x
```

## Usage

Paste an X or Twitter URL on its own line in a note:

```
https://x.com/user/status/1234567890
```

The post will be rendered as an embedded tweet in the preview.

### Link format

When you paste an X or Twitter URL, the plugin offers an "X embed iframe" format option that inserts the embed as raw HTML.

### Supported URLs

- `https://x.com/...`
- `https://twitter.com/...`
